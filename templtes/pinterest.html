Pinterest Redesign

Overview

Introduction


This project focuses on redesigning the Pinterest app to improve user experience, navigation, and visual appeal. The goal is to create a more intuitive and modern interface that enhances accessibility and engagement.

Problem Statement


  •   The design needs modernization to align with current UI/UX trends.
  •   There is a need for a more cohesive and engaging color scheme.
  •   Key interface elements require improvements for better usability and navigation.

Objective


  •   Modernize the layout for better aesthetics.
  •   Enhance the visual appeal of Pinterest by integrating a more cohesive design.
  •   Optimize navigation and usability through targeted design improvements.

Features

Home Page

•   Changed the navigation bar color to red, matching the Pinterest icon.
•   Enhances visual appeal, making the design more eye-catching and
      attractive.
•   Modified the navigation bar by adding a "Create Post" button for
      easier content creation.
•   Improved notification readability by removing unnecessary
      notifications.
•   Retained only relevant notifications, such as likes and comments.
•   Improved visibility of pins from the user's following list.
•   Added a "Following" tab to display only pins from followed users.
•   Included the username of the pin's creator for better clarity.
•   Removed unrelated pins from the home page feed.
•   Uploading pins was complicated
      and required multiple steps.
•   The old upload button was only
      only accessible from the profile.
•   Moved the upload button to the
      center of the navigation bar.

Pin Interface

•   The old interface lacked a button to view more
      related pins.
•   Added a "Show More" button at the bottom to
      display additional related pins.
•   Limited interaction features in the old interface,
      with only pins shown.
•   Add a like button to enhance user interaction.
•   Rearranged buttons for better visibility.

Message Interface

•   The old interface only allowed one-on-one messaging.
•   Added a group messaging feature where users can create groups by
      adding contacts or followers.
•   Previously, there was no option to share pins through messages.
•  Added a pin button in each chat to allow users to send pins directly.

Search Interface

•   Many users preferred a speech recognition feature for searching.
•   Added a speech search button in the search bar.
•   Allows users to search using voice commands.

Profile Interface

•   The old interface lacked an edit profile button.
•   Added an edit profile button in the new interface.
•   Allows users to easily update their profile picture
      and biography.

Methodologies

Survey & Data Collection


  •   Created a questionnaire to gather feedback on the Pinterest app's design and usability.
  •   Focused on aspects like aesthetic cohesiveness and user experience.

Usability Evaluation (SUS Analysis)


  •   Applied the System Usability Scale (SUS) to assess survey responses.
  •   The survey yielded an average SUS score of 58 (below the standard average of 68).
  •   Analysis showed most participants rated the app’s usability as poor (Grade D).

Prototype Development


  •   Create the protoype of the improved Pinterest interface using JustInMind.
  •   Incorporated user feedback into the redesign, improving navigation, aesthetics, and features.

Project Documentation


  •   Compiled findings, analysis, and redesign details into a final project report.
