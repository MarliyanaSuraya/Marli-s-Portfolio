Synthesia Chatbot

Overview

Introduction


The project involves creating a chatbot for Synthesia's Google Chat, utilizing RocheWiki FAQs as the primary knowledge base. By integrating DialogFlow and Vertex AI, the chatbot delivers accurate responses while ensuring reliability through cited sources. This enhances user support by providing quick and precise answers to queries.

Problem Statement


In the Synthesia community, there exists a recurring issue where users repeatedly ask the same questions, leading to redundancy in responses from the support team. This inefficiency results in a waste of resources and time. To address this challenge, there is a need for an automated solution that can efficiently handle common queries, provide accurate information sourced from the appropriate sites, and guide users to relevant resources.

Objective


To integrate a chatbot into the existing Synthesia chat community to provide instant responses by leveraging RocheWiki Synthesia FAQs. If the user's question is not covered in the FAQs, the chatbot will utilize the Vertex AI Grounding feature to conduct a Google search and provide an appropriate answer.

Features

Functional Requirements

  • Automated Responses
    • Upon receiving a user query, the chatbot should provide automated responses based on the relevant information extracted from the RocheWiki Synthesia FAQs.
    • The chatbot should prioritize RocheWiki Synthesia FAQs when answering questions.
  • Fallback Mechanism
    • If the user's question is not found in RocheWiki Synthesia FAQs, the chatbot must use the Vertex AI Grounding feature to perform a Google search and fetch appropriate answers.
    • The chatbot must ensure that the fallback search results are relevant and contextually appropriate.
  • Chatbot Integration
    • The chatbot should seamlessly integrate into the Synthesia chat community platform.
    • It should be accessible to users within the chat interface, available 24/7 to provide assistance.
  • Error Handling
    • The chatbot should handle unexpected user inputs gracefully and provide appropriate fallback messages.
    • The chatbot should log errors for further analysis and improvement.
