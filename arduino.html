RFID Door Lock

Overview

Introduction


With the rise in burglary cases, home security has become a major concern, making traditional locks less reliable. This project introduces an RFID-based smart door lock system built using Arduino for the hardware and programmed in C++ for serial communication. By leveraging RFID technology, the system enhances security and convenience by enabling keyless entry while preventing unauthorized access. This approach provides a modern, scalable, and efficient solution for homes, offices, and businesses.

Problem Statement


  • Increasing burglary rates highlight the need for advanced security solutions.
  • Traditional locks are vulnerable to unauthorized access and key duplication.
  • Need of a more convenient and secure way to control door access without relying on physical keys.

Objective


  • Develop an RFID-based smart door lock system using Arduino and C++.
  • Enable keyless entry using RFID technology for homes, offices, and businesses.
  • Ensure secure access control by preventing unauthorized duplication of RFID cards.
  • Provide a scalable and reprogrammable solution to adjust access permissions as needed.

Features

  • RFID Authentication:   Reads the UID tag of the RFID card when scanned on the RFID Reader Module.
  • Access Control:   Grants access if the scanned UID matches the authorized UID stored in the code; denies access otherwise.
  • LCD Display Feedback:   Displays a message on the LCD screen indicating whether access is granted or denied.
  • Arduino-Based Operation:   Uses an Arduino Uno R3 Board to process RFID data and control access.
  • Keyless Entry:   Eliminates the need for traditional keys, providing a more secure and convenient access method.
  • Real-Time Response:   Instantly verifies the RFID card and provides immediate feedback on the LCD display.

                                                                                                                                                                                                                                                                 

Methodologies

Component Setup


  • Choose the necessary hardware, including Arduino Uno R3, RFID Reader Module, RFID card, LCD display, breadboard, and wires.
  • Install Arduino IDE for programming and uploading code to the Arduino board.

Circuit Assembly


  • Connect the RFID Reader Module to the Arduino Uno R3, ensuring proper wiring to ground, 5V, and digital pins (9, 10, 11, 12).
  • Connect the LCD display screen to the Arduino, linking ground, 3.5V, analog input 4, and analog input 5.

RFID Authentication


  • Write and upload a C++ program using Arduino IDE to read the UID tag from the RFID card.
  • Implement logic to compare scanned UID with predefined authorized UIDs stored in the code.
  • Display access status ("Access Granted" or "Access Denied") on the LCD screen.

Testing & Validation


  • Scan an authorized RFID card and check if access is granted with a correct LCD message.
  • Scan an unauthorized RFID card and ensure access is denied with the appropriate LCD message.
  • Verify that all components function correctly and debug any errors in circuit connections or code.
